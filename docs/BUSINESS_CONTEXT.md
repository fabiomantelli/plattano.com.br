### 3. Storage Solutions

- **Type:** Backup storage appliance (on-premises)
- **Features:**
  - Up to 95% deduplication
  - Ultra-fast performance
  - Linear scalability

  - Native Veeam integration
- **Target:** Large Veeam environments (>10TB), on-premises infrastructure

#### Plattano Hot Storage
- **Type:** S3-compatible cloud storage
- **Backend:** Wasabi
- **Features:**
  - 100% S3 API compatible
  - Integrated ransomware protection
  - Transparent pricing
  - 99.9% SLA
  - Multi-region availability
- **Target:** Cloud-first workloads, off-site backup, unpredictable growth
